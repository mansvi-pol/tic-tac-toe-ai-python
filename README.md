# Tic-Tac-Toe AI using Python

This project is an AI-powered Tic-Tac-Toe game developed using Python as part of the Python & Artificial Intelligence from A-Z Bootcamp by Devtown in 2024.

The project was initially developed as a multiplayer game where two players could compete against each other. It was later enhanced by me implementing the Minimax Algorithm, allowing the computer to analyze possible moves and make optimal decisions while playing against the user.

## Key Features

- Single Player Mode (Player vs AI)
- Multiplayer Mode (Player vs Player)
- Minimax Algorithm implementation
- Win, Loss, and Draw detection
- Interactive console-based gameplay
- Strategic AI decision making

## Skills Applied

- Python Programming
- Functions and Loops
- Problem Solving
- Artificial Intelligence Fundamentals
- Minimax Algorithm
- Game Development Logic

This project helped strengthen my Python fundamentals while providing hands-on experience in implementing AI concepts through a practical game development project.

## How to Run

- Download or clone the repository
- Open the project folder in your preferred Python environment.
- Run the Python file:
  python tictactoe.py
- Choose:
  1. for Single Player (Player vs AI)
  2. for Multiplayer (Player vs Player)
- Follow the on-screen instructions to play the game.
